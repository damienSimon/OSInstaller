Guide d'installation automatisé du poste de dev sous Ubuntu 18.04

ouvrir un terminal et se connecter en root via "sudo -i"
lancer le script d'installation:
". install_complete_poste_dev_ubuntu_18_04.sh"

Une fois terminé, l'ordinateur va redemarrer.
ouvrir un terminal et se connecter en root via "sudo -i"
Lancer à present le script de configuration :
". config_ubuntu.sh"
Une fois terminé, l'ordinateur va redemarrer.

Si vous le souhaitez vous pouvez installer minikube.
Pour cela, ouvrir un terminal, se connecter en root via "sudo -i".
Lancer la commande suivante :
". install_minikube.sh"

A présent, passer à la partie "manuelle" pour finaliser l'installation.
